## Deskripsi Aplikasi
Aplikasi manajemen produk berbasis Flutter dengan fitur CRUD menggunakan SQLite, login lokal, session, dan integrasi WhatsApp.

## Fitur
- Splash Screen
- Login & Session
- CRUD Produk (SQLite)
- ListView & GridView
- Detail Produk
- Redirect WhatsApp
- Logout

## Teknologi
- Flutter
- Dart
- SQLite (sqflite)
- SharedPreferences
- url_launcher

## Cara Menjalankan
1. flutter pub get
2. flutter run
